# Hyperswitch on Railway

Complete Hyperswitch payment system deployment on Railway cloud platform.

## Features
- ✅ Zero port conflicts
- ✅ Automatic PostgreSQL + Redis
- ✅ HTTPS endpoints
- ✅ Global deployment
- ✅ Free tier available

## Quick Deployment
1. Clone this repository
2. Go to [Railway](https://railway.app/dashboard)
3. New Project → Deploy from GitHub repo
4. Select this repo
5. Railway auto-detects configuration
6. Deploy!

## Configuration
- `railway.toml` - Complete Railway setup
- `dockerfile.railway` - Optimized Docker
- `hyperswitch-config.toml` - Server config
- `.env.template` - Environment variables

## Features Included
- Payment gateway connectors (Stripe, PayPal, etc.)
- Admin dashboard
- API endpoints
- Database management
- SSL/HTTPS

## After Deployment
- Access your Railway URL for API
- Configure payment connectors
- Update security keys in Railway Variables
